# NAME

**pathctl.bash** - Utility for PATH management

# SYNOPSYS

    #!bash
    source pathctl.bash
    pathctl_push    /path/to/your-bin
    pathctl_unshift /path/to/your-bin

    # show verbose messages
    PATHCTL_VERBOSE=1

# DESCRIPTION

Add functions to manage PATH variable.
